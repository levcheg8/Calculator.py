# Calculator.py
Консольный калькулятор на Python с накоплением результата. Поддерживает основные операции: +, -, *, /, //, %, ^, root. Мой первый проект на Python.
# 🧮 Calculator — Мощный калькулятор на Python

> Простой и удобный калькулятор с графическим интерфейсом.

---

## ✨ Возможности

- ➕ Сложение, вычитание, умножение, деление
- 📐 Возведение в степень, квадратный корень, проценты
- 🖱️ Удобные кнопки + поддержка клавиатуры
- ⚡ Один `.exe` файл — не требует установки

---

## 🚀 Как запустить

1. Найдите файл `calculator.exe` в корне репозитория
2. Скачайте его (нажмите на файл → **Download** или **View raw**)
3. Запустите двойным кликом
4. Готово!

> ⚠️ Если Windows покажет предупреждение — нажмите **"Подробнее"** → **"Выполнить в любом случае"**.
