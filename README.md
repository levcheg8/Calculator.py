# Calculator.py
Консольный калькулятор на Python с накоплением результата. Поддерживает основные операции: +, -, *, /, //, %, ^, root. Мой первый проект на Python.
# Calculator.py

Мой первый проект на Python — консольный калькулятор с накоплением результата.  
Со временем оброс графическим интерфейсом и превратился в полноценную программу.

---

## Что умеет

- Базовые операции: `+`, `-`, `*`, `/`
- Целочисленное деление `//`, остаток `%`
- Степень `^` и квадратный корень `root`
- Удобные кнопки + можно вводить с клавиатуры
- Накопление результата — считает как калькулятор в телефоне

---

## Как запустить

В папке репозитория лежит `calculator.exe`:

1. Скачайте его (нажмите на файл → **Download**)
2. Запустите двойным кликом
3. Пользуйтесь 

*Если Windows ругается — нажмите "Подробнее" → "Выполнить в любом случае". Это нормально для самодельных .exe.*
